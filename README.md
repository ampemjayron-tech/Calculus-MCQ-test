### Running the quiz

- **Locally:** Open `excel-chapter1-mcq.html` in any modern browser вҖ” no server needed.
- **GitHub Pages:** Once Pages is enabled (Settings вҶ’ Pages вҶ’ Deploy from branch `main`, root `/`), the quiz is accessible at `https://<username>.github.io/Calculus-MCQ-test/excel-chapter1-mcq.html`.

### Features

| Feature | Details |
|---------|---------|
| **Quiz modes** | **Practice** вҖ“ immediate feedback + explanation after each answer; **Exam** вҖ“ all feedback deferred until end |
| **Topic filtering** | Choose any subset of the 13 topics below |
| **Randomisation** | Toggle shuffle for questions and/or answer options |
| **Score tracking** | Live score in Practice mode; summary pie-style ring chart at end |
| **Review screen** | See all questions, your answer, the correct answer, and the explanation; filterable by All / Wrong+Skipped / Correct |
| **Keyboard navigation** | `1`вҖ“`4` select options В· `Enter`/`Space` confirm/next В· `вҶҗ`/`вҶ’` navigate |
| **Accessibility** | ARIA roles, `aria-label` on every interactive element, `aria-live` for feedback |
| **Responsive** | Works on mobile, tablet, and desktop |
